DragonOS is a custom privacy-hardened penetration testing operating system. No Google. No bloatware. Pre-configured security tools, automated launcher, and optional full Arch Linux desktop. The Dragon Phone series runs DragonOS — three tiers built for any skill level.

DEVICE LINEUP

MOBILE — Dragon Phone Series
Dragon (Tier 1): Rootless DragonOS. Terminal tools via NetHunter proot. Entry level device for learning penetration testing basics. No root required.

Dragon+ (Tier 2): Rooted with custom NetHunter kernel. Automated Dragon Deploy menu launcher. All CLI tools pre-installed. WiFi monitor mode and packet injection available with USB OTG adapter.

Dragon Pro (Tier 3): Everything in Dragon+ with USB OTG adapter included. DragonOS app delivers a full Arch Linux KDE Plasma tablet environment with BlackArch security repos.

LAPTOP — Dragon Book Series
Standard and Pro models. Arch Linux with KDE Plasma. BlackArch repos. Full Dragon tool suite pre-installed. Encrypted storage.

DESKTOP — Dragon Station Series
Workstation and Lab server builds. DragonOS with complete BlackArch tool suite. Optimized for Hashcat GPU cracking, Metasploit operations and security lab environments.

DRAGONOS BUILDS
DragonOS Mobile: Based on LineageOS/Android with NetHunter. Magisk root. MicroG privacy layer. Automated Dragon Deploy launcher. Optional full Arch Linux via DragonOS app.

DragonOS Desktop: Based on Arch Linux with KDE Plasma. BlackArch security repos. Full Dragon tool suite pre-installed. Encrypted storage. Hardened kernel.
